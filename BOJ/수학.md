### 2609번
#### 최대공약수와 최소공배수

#### 문제
https://www.acmicpc.net/problem/2609

#### 코드

``` python
N, M = map(int, input().split())

check = min(N, M)

common_div = 0
# 최대공약수
for i in range(1, check+1):
    if N % i == 0 and M % i == 0:
        common_div = i

common_mul = 0
# 최소공배수
common_mul = common_div * (N // common_div) * (M // common_div)

print(common_div)
print(common_mul)
```
