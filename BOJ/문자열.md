### 2675번
#### 문자열 반복

#### 문제
https://www.acmicpc.net/problem/2675

#### 코드

``` python
T = int(input())

answer = []

for _ in range(T):
    R, S = input().split()
    
    P = ''
    for i in S:
        P += i * int(R)
    answer.append(P)
    
for a in answer:
    print(a)
```

### 10809번
#### 알파벳 찾기

#### 문제
https://www.acmicpc.net/problem/10809

#### 코드

``` python
S = input()

answer = [-1]*26    # -1로 초기화

for i in range(len(S)): # 아스키코드로 변환해서 a를 0번 인덱스부터 채움
    if answer[ord(S[i])-97] != -1:  # -1이 아니면 이미 처음 나온 문자의 위치를 담은 것이므로 continue
        continue
    else:
        answer[ord(S[i])-97] = i

for r in answer:
    print(r, end=' ')
```
