### 2231번
#### 분해합

#### 문제
https://www.acmicpc.net/problem/2231

#### 코드

``` python
N = int(input())

for i in range(1, N+1):
    answer = 0
    num_list = list(map(int, str(i)))
    answer += i + sum(num_list)
    if answer == N:
        print(i)
        break
    if i == N:
        print(0)
```
