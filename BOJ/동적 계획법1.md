### 2748번
#### 피보나치 수 2

#### 문제
https://www.acmicpc.net/problem/2748

#### 코드

``` python
import sys

n = int(sys.stdin.readline())

fibo_list = [0 for _ in range(n+1)]

fibo_list[1] = 1

for i in range(2, n+1):
    fibo_list[i] = fibo_list[i-1] + fibo_list[i-2]

print(fibo_list[-1])
```
